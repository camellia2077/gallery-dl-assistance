## 感谢与依赖 (Acknowledgements & Dependencies)

本项目依赖于强大的开源下载工具 **gallery-dl** 来实现主要的媒体内容获取功能。

### 核心依赖

| 工具 | 描述 | 链接 |
| :--- | :--- | :--- |
| **gallery-dl** | 一个命令行程序，用于从各种图库和图像主机下载图像，我们用它来下载 [Bilibili网站] 的媒体内容。 | [GitHub: gallery-dl](https://github.com/mikf/gallery-dl) |
